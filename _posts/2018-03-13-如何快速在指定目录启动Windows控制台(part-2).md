## 优化方案:添加加速键

根据菜鸡作者的Qt平台编辑经验，添加加速键只要在**Action**的**Name**里面添加

```
&[Key]
```

那么此Action的加速键就是Key

为了避免加速键冲突，我选择了键盘左上角的~/`键作为加速键。同时为了美观，在加速键左右加上括号

所以把cmdPrompt的“默认”属性更改为

```
CMD Here(&`)
```

或者

```
CMD Here(&~)
```
